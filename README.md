# 설 연휴 휴무 확인 도구

가게 목록을 업로드하고, 네이버지도에서 설 연휴 휴무 등록 여부를 확인하는 도구입니다.

## 사이트 주소

**https://yundain.github.io/holiday-checker/**

## 주요 기능

| 기능 | 설명 |
|------|------|
| 엑셀 업로드 | xlsx, xls, csv 파일 지원 |
| 직접 입력 | 가게명을 한 줄씩 입력 |
| 네이버지도 연동 | 클릭 시 해당 가게 검색 페이지로 이동 |
| 상태 체크 | 휴무 / 정상영업 버튼으로 기록 |
| 메모 | 가게별 메모 작성 |
| 필터 | 전체 / 미확인 / 휴무 / 정상영업 |
| 진행률 | 확인 완료율 표시 |
| 결과 내보내기 | 엑셀 파일로 다운로드 |
| 자동 저장 | 브라우저를 닫아도 데이터 유지 |

## 사용 방법

### 1. 가게 목록 추가

**방법 A: 엑셀 파일 업로드**
- xlsx, xls, csv 파일을 드래그 앤 드롭
- 첫 번째 열에 가게명이 있어야 함

**방법 B: 직접 입력**
- 텍스트 입력란에 가게명을 한 줄에 하나씩 입력
- "가게 추가" 버튼 클릭

### 2. 휴무 확인

1. "네이버지도" 버튼 클릭 → 새 탭에서 검색 결과 확인
2. 영업시간/휴무일 정보 확인
3. "휴무" 또는 "정상영업" 버튼 클릭하여 기록
4. 필요시 메모 입력

### 3. 결과 내보내기

- "결과 내보내기" 버튼 클릭
- `휴무확인결과_YYYY-MM-DD.xlsx` 파일 다운로드

## 데이터 저장 방식

### LocalStorage 사용

데이터는 **브라우저의 LocalStorage**에 저장됩니다.

```
키: holidayChecker
값: JSON 배열
```

#### 저장되는 데이터 구조

```json
[
  {
    "id": 1736668800000.123,
    "name": "브레댄코 상왕십리점",
    "status": "holiday",
    "memo": "1/28~1/30 휴무"
  },
  {
    "id": 1736668800000.456,
    "name": "스타벅스 강남역점",
    "status": "normal",
    "memo": ""
  }
]
```

#### 특징

| 항목 | 설명 |
|------|------|
| 저장 위치 | 사용자의 브라우저 (로컬) |
| 용량 제한 | 약 5MB (브라우저별 상이) |
| 서버 전송 | 없음 (완전 로컬) |
| 개인정보 | 서버에 저장되지 않음 |
| 데이터 유지 | 브라우저 캐시 삭제 전까지 유지 |
| 기기 간 공유 | 불가 (해당 브라우저에서만 접근) |

#### 주의사항

- 브라우저 캐시/데이터 삭제 시 데이터도 삭제됨
- 다른 기기나 브라우저에서는 데이터가 공유되지 않음
- 중요한 데이터는 "결과 내보내기"로 엑셀 백업 권장

## 기술 스택

- LocalStorage - 데이터 저장
- GitHub Pages - 호스팅
